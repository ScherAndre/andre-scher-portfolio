## ✨ Funcionalidades

- **Layout responsivo** — funciona em mobile, tablet e desktop
- **Efeito de typewriter** — animação de texto no hero com JavaScript puro
- **Scroll reveal** — elementos aparecem com animação ao rolar a página (Intersection Observer API)
- **Dark mode** — tema escuro com paleta de cores coesa em CSS variables
- **Sem frameworks** — HTML, CSS e JS puro, sem dependências externas
- **Seções completas** — Hero, Sobre, Skills, Projetos e Contato
